## How run

bash snake.sh </br>
bash tetris.sh

## Available commands

s = move down <br/>
w = move up <br/>
a = move left <br/>
d = move right <br/>
q = quit <br/>

<br/>
(tetris only) </br>
k = rotate right <br/>
l = rotate left <br/>
